# TkPy3一个升级版的TkPy IDE
# 安装TkPy3:
```
pip install tkpy3
```

## 安装依赖
```

pip install PyQt5
python -m TkPy3.tkpy3_tools.relys
```
## 运行TkPy3
> 如果已安装TkPy3
```
python -m TkPy3
```

> 没有安装
```
python run_code.py
```
> ## 注意: 不要用IDLE运行
